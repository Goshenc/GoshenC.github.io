SQLite用法

Created2025-03-15|Updated2025-03-15
|Word Count:1.2k|Reading Time:5mins|Post Views:

在 Android 中，SQLite 是一个轻量级的关系型数据库，它允许应用本地存储和管理数据。你可以使用 SQLite 来保存、查询、更新、删除应用数据。下面是如何在 Android 中使用 SQLite 的步骤。

1. 创建数据库助手类（SQLiteOpenHelper）

首先，你需要创建一个继承自 SQLiteOpenHelper 的类，这个类用于管理数据库的创建和版本管理。SQLiteOpenHelper 会自动处理数据库的打开、关闭、更新等操作。

2. 打开数据库

在你的应用中使用数据库时，你可以通过 SQLiteOpenHelper 类来打开数据库，获取 SQLiteDatabase 实例：

  • writableDatabase：获取可写的数据库（用于插入、更新和删除数据）。
  • readableDatabase：获取只读的数据库（用于查询数据）。

3. 插入数据

你可以通过 SQLiteDatabase 对象的 insert() 方法插入数据。这个方法要求你提供表名和要插入的列名-值对。

  • ContentValues 用于存储列名和对应的值。
  • insert() 方法返回新插入行的 ID。

4. 查询数据

使用 SQLiteDatabase 对象的 query() 或 rawQuery() 方法来查询数据。

  • query() 方法返回一个 Cursor 对象，它是数据库查询结果的表示。
  • 通过 cursor.getColumnIndex() 获取列的索引，然后通过相应的 get 方法获取数据。

5. 更新数据

使用 update() 方法来更新数据库中的数据：

  • update() 方法会更新符合条件的行，selection 是查询条件，selectionArgs 是条件的参数。

6. 删除数据

你可以使用 delete() 方法来删除数据：

  • delete() 方法会删除符合条件的行，selection 是查询条件，selectionArgs 是条件的参数。

总结

  • SQLiteOpenHelper：管理数据库的创建、升级等操作。
  • SQLiteDatabase：用于执行数据库操作，如插入、查询、更新、删除。
  • ContentValues：用于存储要插入或更新的数据。
  • Cursor：用于访问查询结果。

SQLite 是一种非常有效的方式来在 Android 上进行本地数据存储，尤其适用于存储小型结构化数据。
